# build-your-own-lisp

Following https://www.buildyourownlisp.com/

## Compile

`cc -std=c99 -Wall hello_world.ic -ledit -o hello_world`

### Windows

`./compile.sh <filename>`

### Linux

`bash compile.sh <filename>`
